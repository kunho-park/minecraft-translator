# AI 번역기 사용 가이드

## 1. 강력 추천: Google Cloud 무료 크레딧 활용

**Google Cloud에 가입하여 $300(약 40만원)의 무료 크레딧을 받는 것을 강력하게 추천합니다.**

기본 무료 계정은 사용량 제한이 낮아 번역 속도가 느리거나 중간에 멈출 수 있습니다. Google Cloud의 $300 무료 크레딧을 받으면, 이 크레딧을 사용하여 비용 부담 없이 훨씬 빠른 속도(높은 RPM/TPM)로 쾌적하게 번역할 수 있습니다.

## 2. API 키 발급 방법

Google AI Studio에서 손쉽게 API 키를 발급받고, Google Cloud 프로젝트와 연동하여 무료 크레딧을 사용할 수 있습니다.

1. **Google Cloud 가입 및 크레딧 받기**: [Google Cloud Console](https://console.cloud.google.com/)에 접속하여 가입하고 결제 수단을 등록하여 $300 무료 크레딧을 받습니다.
2. **Google AI Studio 접속**: [Google AI Studio](https://aistudio.google.com/)로 이동합니다.
3. **API 키 생성**: 'Get API key' 버튼을 클릭합니다.
4. **프로젝트 연동 (중요)**: 키를 생성할 때 **'Create API key in existing project'**를 선택하고, 앞서 1번에서 만든(크레딧이 있는) Google Cloud 프로젝트를 선택합니다. 이렇게 해야 무료 크레딧이 차감되며 높은 성능 제한으로 이용할 수 있습니다. (경우에 따라 유료 계정을 활성화해야. Tier 1로 바뀜.)

## 3. 다운로드 및 실행

1. **다운로드**: 배포된 최신 버전의 `AutoTranslate-X.X.X-windows.exe`압축 파일을 다운로드합니다. ([다운로드로 이동](https://github.com/kunho-park/minecraft-translator/releases))
2. **실행**: `AutoTranslate-X.X.X-windows.exe`(또는 실행 파일)을 더블 클릭하여 프로그램을 실행합니다.
3. **모드팩 준비**: 현재 Prism Launcher 혹은 CurseForge 만 간편 선택 기능 지원합니다. (그외 런처도 수동 지정 가능)

## 4. 용어 설명 및 설정 가이드

프로그램 설정 시 마주하게 될 용어들에 대한 설명입니다.

* **모델 (Model)**: 번역을 수행할 AI 두뇌입니다.
* **온도 (Temperature)**: AI의 창의성을 조절합니다. `0`으로 설정하면 가장 정확하고 일관된 번역을 합니다.
* **배치 사이즈 (Batch Size)**: 한 번에 AI에게 묶어서 보낼 문장 수입니다.
* **동시 실행 (Concurrent executions)**: 동시에 작업을 진행할 개수입니다.
* **RPM (Requests Per Minute)**: 1분당 요청 횟수 제한입니다.
* **TPM (Tokens Per Minute)**: 1분당 처리할 글자(토큰) 수 제한입니다.

## 5. 추천 세팅 (Best Practice)

Google Cloud 무료 크레딧 사용자를 위한 **최적의 속도와 가성비 세팅**입니다.

| 설정 항목             | 추천 값               |
| :-------------------- | :-------------------- |
| **Model**       | `gemini-2.0-flash`  |
| **Temperature** | `0`                 |
| **Batch Size**  | `100`               |
| **동시 실행**   | `45`                |
| **RPM**         | `1500`              |
| **TPM**         | `2,000,000` (200만) |

## 6. 모드팩 번역 진행 방법

1. **모드팩 선택**: 번역하려는 모드팩의 경로를 지정합니다.
2. **스캔**: 번역이 필요한 파일들을 스캔합니다.
3. **설정 입력**: 위에서 발급받은 API 키와 **추천 세팅** 값을 입력합니다.
4. **번역 시작**: 번역을 시작합니다.

## 7. 팁

* **용어집(Glossary) 사용 (권장)**: 마인크래프트 전용 용어집을 사용하면 아이템 이름 등이 공식 번역과 일치하게 번역되어 퀄리티가 훨씬 좋아집니다. 웬만하면 사용하는 것을 추천합니다.
* **리뷰(Review) 기능**: 번역된 내용을 AI가 다시 검토하는 기능입니다. 퀄리티는 조금 더 좋아질 수 있으나, **시간이 매우 오래 걸립니다**. 퀄리티가 정말 중요한 경우가 아니라면 끄고 진행하는 것이 정신건강에 좋습니다.
